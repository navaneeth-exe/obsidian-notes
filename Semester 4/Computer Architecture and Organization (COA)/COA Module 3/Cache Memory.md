## **Definition**

Cache memory is a **small, high-speed memory** located close to the CPU that stores **frequently used data and instructions** for faster access.

---

## **Basic Concept**

- CPU is very fast, main memory is slower
- Cache is placed between **CPU and main memory**
- It reduces the **time taken to access data**

---

## **Working**

1. CPU requests data
2. First checks **cache memory**
3. If found → **cache hit (fast access)**
4. If not → **cache miss**
5. Data is fetched from main memory and stored in cache

---

## **Characteristics**

- Very fast access time
- Small size
- Expensive
- Usually built **inside or near CPU**

---

## **Types of Cache**

- **L1 Cache** → fastest, smallest
- **L2 Cache** → larger, slower than L1
- **L3 Cache** → larger, shared among cores

---

## **Advantages**

- Reduces memory access time
- Improves system performance
- Minimizes CPU waiting time
