## **Definition**

An **embedded system** is a **dedicated computer system** designed to perform a **specific task** and control operations within a larger system.

## **Basic Concept**

- It is a combination of **hardware and software**
- Designed for **specific functions only (not general-purpose)**
- Commonly used in electronic and control systems

---

## **Components of Embedded System**

1. **Processor / Microcontroller**
    - Acts as the brain
    - Executes instructions and processes data
2. **Memory**
    - Stores program and data
3. **Input Devices**
    - Provide data to the system
    - Example: sensors, switches
4. **Output Devices**
    - Display or act on processed data
    - Example: LED, display, motor
5. **Software**
    - Program that controls system operation

---

## **Examples**

- Washing machine
- Microwave oven
- Traffic light controller
- Digital camera
