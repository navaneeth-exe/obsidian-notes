### Definition

The **Central Processing Unit (CPU)** is the brain of the computer. It performs **data processing**, **logical operations**, and **controls all activities** of the computer system.

---

## Main Components of CPU

The internal architecture of a CPU mainly consists of:

1. **Arithmetic Logic Unit (ALU)**
2. **Control Unit (CU)**
3. **Registers**
4. **Internal Buses**

---

## 1. Arithmetic Logic Unit (ALU)

- Performs **arithmetic operations** like addition, subtraction, multiplication, division.
- Performs **logical operations** like AND, OR, NOT, XOR.
- Also handles **comparison operations** (>, <, =).

**Functions of ALU:**

- Integer calculations
- Bitwise operations
- Shift and rotate operations

📌 _Exam keyword:_  
👉 “ALU is responsible for all arithmetic and logical processing inside the CPU.”

---

## 2. Control Unit (CU)

- Acts as the **traffic police** of the CPU 🚦
- Controls and coordinates all operations of the CPU.
- Fetches instructions from memory and decodes them.
- Sends control signals to ALU, registers, memory, and I/O devices.

**Functions of CU:**

- Instruction fetch
- Instruction decoding
- Timing and control
- Sequencing of operations

📌 _Exam keyword:_  
👉 “Control Unit does not process data but directs the operation of the processor.”

---

## 3. Registers

Registers are **small, high-speed memory locations** inside the CPU used to store temporary data, instructions, and addresses.

### Important Registers (VERY IMPORTANT FOR EXAM ⚠️)

- **Program Counter (PC)**  
    Stores the address of the next instruction to be executed.
- **Instruction Register (IR)**  
    Holds the current instruction being executed.
- **Accumulator (ACC)**  
    Stores intermediate results of ALU operations.
- **Memory Address Register (MAR)**  
    Holds the address of the memory location to be accessed.
- **Memory Data Register (MDR)**  
    Holds data being transferred to or from memory.
- **General Purpose Registers**  
    Used for temporary storage during execution.

📌 _Exam keyword:_  
👉 “Registers provide fastest data access to the CPU.”

---

## 4. Internal Buses

Buses are communication paths inside the CPU.

### Types of Buses:

- **Data Bus** – Transfers data
- **Address Bus** – Transfers memory addresses
- **Control Bus** – Transfers control signals

📌 _Exam keyword:_  
👉 “Buses enable communication between CPU components.”
