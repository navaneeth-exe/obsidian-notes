## **Definition**

An assembler is a system software that **converts assembly language programs into machine language (binary code)**.

---

## **Basic Concept**

- Assembly language uses **mnemonics (ADD, MOV, etc.)**
- CPU understands only **machine language (0s and 1s)**
- Assembler performs this **translation**

---

## **Functions of Assembler**

1. Converts **mnemonic instructions → binary code**
2. Assigns **memory addresses to instructions**
3. Handles **labels and symbols**
4. Detects **errors in the program**

---

## **Working**

- Input: Assembly language program
- Process: Translation by assembler
- Output: Machine code
