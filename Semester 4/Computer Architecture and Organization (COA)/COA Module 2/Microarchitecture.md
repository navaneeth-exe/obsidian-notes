## **Definition**

Microarchitecture refers to the **internal organization and implementation of a processor**, describing how the CPU components work together to execute instructions.

---

## **Basic Concept**

- It defines **how a processor is designed internally**
- Implements the **instruction set architecture (ISA)**
- Deals with **hardware-level details**

---

## **Components of Microarchitecture**

1. **ALU (Arithmetic Logic Unit)**
    - Performs arithmetic and logical operations
2. **Registers**
    - Store data temporarily during execution
3. **Control Unit**
    - Controls and coordinates all operations
4. **Data Path**
    - Path through which data flows inside CPU
5. **Cache Memory**
    - Provides fast access to frequently used data

---

## **Working**

- Instruction is fetched from memory
- Decoded by control unit
- Executed using ALU and registers
- Result is stored back
