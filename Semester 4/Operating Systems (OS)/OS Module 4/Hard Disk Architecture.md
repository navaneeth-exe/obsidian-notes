## **Definition**

A **hard disk** is a secondary storage device used to **store data permanently** using magnetic storage.

![[Pasted image 20260412013058.png|183]]

## **Main Components**

### **1. Platters**

- Circular disks coated with magnetic material
- Store actual data

---

### **2. Spindle**

- Rotates the platters at high speed (e.g., 5400/7200 RPM)

---

### **3. Read/Write Head**

- Reads and writes data
- Moves over platter surface

---

### **4. Actuator Arm**

- Moves the read/write head to correct position

---

### **5. Tracks**

- Circular paths on platter where data is stored

---

### **6. Sectors**

- Smallest unit of storage on disk
- Each track is divided into sectors

---

### **7. Cylinders**

- Set of tracks at same position across all platters

---

## **Working of Hard Disk**

1. Disk spins continuously
2. Actuator moves head to correct track
3. Head reads/writes data on sector
4. Data transferred to/from memory

---

## **Addressing (IMPORTANT)**

- Data located using:
    - Track number
    - Sector number
    - Cylinder number

---

## **Key Points (WRITE FOR MARKS)**

- Magnetic storage device
- Uses rotating platters
- Data organized as tracks and sectors
- Head movement determines access time
